# SEPA IBAN Generator & Validator 💳

Полностью клиентское веб-приложение для генерации и валидации IBAN-счетов в стандарте SEPA. 
Приложение работает прямо в браузере (Vanilla HTML, CSS, JS), не требует бэкенда и готово к деплою на **GitHub Pages**.

## 🌟 Возможности

1. **Генерация IBAN**:
   - Автоматический расчет контрольных цифр (алгоритм MOD-97).
   - Поддержка предустановленных форматов стран: Германия (DE), Франция (FR), Испания (ES), Италия (IT), Нидерланды (NL).
   - Возможность указать собственный двухбуквенный код страны.
   - Настраиваемый код банка (Bank Code) и длина номера счета (Account Number).
   - Генерация от 1 до 100 попыток за один цикл.
   - Настраиваемая задержка между запросами к API для предотвращения блокировок IP.

2. **Валидация**:
   - **Офлайн-проверка**: Мгновенный расчет контрольной суммы MOD-97.
   - **Онлайн-проверка**: Запрос к публичному API [openiban.com](https://openiban.com) для верификации существования счета банком и извлечения информации (BIC, название банка, адрес).
   - Ручной валидатор: Вставьте любой IBAN и получите детальный отчет (корректность MOD-97, статус API, название банка, BIC, адрес).

3. **Интерфейс и экспорт**:
   - Современный футуристичный дизайн с темной темой, эффектом Glassmorphism и плавными анимациями.
   - Фильтрация результатов в реальном времени: "Все", "Живые" (валидные), "Невалидные".
   - Мгновенное копирование в буфер обмена (как отдельных IBAN, так и всего отфильтрованного списка).
   - Экспорт результатов в форматы **TXT** или **JSON**.
   - Адаптивная верстка: приложение одинаково удобно на ПК и мобильных устройствах.

## 🚀 Как запустить локально

Поскольку приложение написано на чистом HTML/CSS/JS без внешних сборщиков, вы можете открыть его локально:

1. С помощью любого локального сервера. Например, в Python:
   ```bash
   python3 -m http.server 8000
   ```
2. Откройте адрес `http://localhost:8000` в браузере.

## 🌐 Деплой на GitHub Pages за 3 шага

Чтобы опубликовать проект на GitHub Pages:

1. **Инициализируйте репозиторий и сделайте коммит**:
   ```bash
   git add .
   git commit -m "Init SEPA Generator Web App"
   ```
2. **Создайте репозиторий на GitHub** и свяжите его с локальным:
   ```bash
   git remote add origin https://github.com/ВАШ_ЛОГИН/НАЗВАНИЕ_РЕПОЗИТОРИЯ.git
   git branch -M main
   git push -u origin main
   ```
3. **Включите GitHub Pages**:
   - Перейдите в настройки репозитория на GitHub (**Settings**).
   - Слева выберите раздел **Pages**.
   - В блоке **Build and deployment** -> **Source** выберите **Deploy from a branch**.
   - В выпадающем списке веток выберите `main` (папка `/ (root)`) и нажмите **Save**.
   
Через 1-2 минуты ваш сайт будет доступен по адресу `https://ВАШ_ЛОГИН.github.io/НАЗВАНИЕ_РЕПОЗИТОРИЯ/`.
